# Glaucoma Detection AI Web Application

## Overview

This web application provides automated glaucoma screening from retinal fundus images using a lightweight deep learning model. Users can upload a retinal image and receive an instant prediction indicating whether glaucoma is detected.

The application is designed as a demonstration of AI-assisted ophthalmic screening and showcases the deployment of a deep learning model in a production-ready web interface.

## Features

* Retinal fundus image upload
* Automated glaucoma prediction
* Real-time inference
* Lightweight deployment architecture
* Accessible through a web browser without local installation

## How It Works

1. Upload a retinal fundus image.
2. The image is preprocessed and resized.
3. The trained deep learning model analyzes the image.
4. The system returns:

   * Diagnosis (Glaucoma / No Glaucoma)
   * Prediction confidence score

## Model Information

The deployed model is based on a lightweight CNN-based glaucoma classification architecture developed for efficient retinal image analysis.

The objective is to support early glaucoma screening by providing rapid automated predictions from fundus photographs.

## Disclaimer

This application is intended for research, educational, and demonstration purposes only.

The predictions generated by the model should not be considered a medical diagnosis and must not replace professional clinical evaluation by qualified ophthalmologists.

## Deployment

This application is deployed using Hugging Face Spaces and Gradio, enabling public access through a simple web interface.
